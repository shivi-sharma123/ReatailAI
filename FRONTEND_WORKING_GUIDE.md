## 🔍 TROUBLESHOOTING

**If port 3000 is in use:**
- Run: `netstat -ano | findstr :3000`
- Kill the process: `taskkill /F /PID [process_id]`

**If npm not found:**
- Install Node.js from nodejs.org
- Restart your computer
- Try again

**If modules missing:**
- Delete `node_modules` folder
- Run `npm install` again
