🔧 Troubleshooting

If frontend is not working:

  1. Open terminal and run: cd client && npm start
  2. Check if port 3000 is available
  3. Make sure Node.js is installed and updated
  4. Try running: npm install if there are dependency issues

Backend issues:

  1. Make sure Python is installed
  2. Run: python client/server/app.py
  3. Check if port 5000 is available
