## 🚀 HOW TO RUN THE APP

### Start Backend (Terminal 1):
```bash
cd client/server
python app.py
```

### Start Frontend (Terminal 2):
```bash
cd client
npm start
```

### Open Browser:
```
http://localhost:3000
```
